### Question 1: What is science?

- **Correct Answer:** C) 5 steps

- **Video Content (q004-incorrect.mp4):** Explain why there are specifically 5 steps, not 3/4/6

### Question 5: Can anyone be a scientist?

## HOW TO CREATE THESE VIDEOS

### Option 1: Using Screen Recording (Easiest)
1. **Open a presentation** (PowerPoint, Google Slides, or document)
2. **Add content:**
   - Question text
   - Answer explanation
   - Diagrams or images
3. **Record your screen** using:
   - Mac: QuickTime (Press Command + Shift + 5)
   - Windows: Game Bar (Press Windows + G)
   - Browser: Built-in recorder extensions
4. **Add narration** while recording (explain the answer clearly)
5. **Export as MP4** to `/Users/abhayindwar/Desktop/examtool/video/`

### Option 2: Using Free Video Editing Software
1. **Download OBS Studio** (Free, all platforms)
2. **Create slides/text** with the answer explanation
3. **Record with narration** (talk while showing slides)
4. **Export as MP4**
5. **Place in `/Users/abhayindwar/Desktop/examtool/video/`**

### Option 3: Using Online Tools
1. **FlexClip** (https://www.flexclip.com/) - Free video editor
2. **Animaker** (https://www.animaker.com/) - Create animated explainer videos
3. **Powtoon** (https://www.powtoon.com/) - Presentation to video
4. **Canva** (https://www.canva.com/video/) - Create videos from templates

## TROUBLESHOOTING

### Video doesn't play
- Check filename is exactly: q00X-correct.mp4 or q00X-incorrect.mp4
- Ensure file is in correct folder: /Users/abhayindwar/Desktop/examtool/video/
- Try different video format (MP4 only)
- Check browser supports video playback (Chrome, Firefox, Safari all support MP4)

### Video file is too large
- Compress using: https://www.freeconvert.com/video-compressor
- Use 720p resolution, not 1080p
- Lower bitrate to 2-3 Mbps

### Audio is too quiet
- Use audio editing tool to increase volume
- Recommended audio level: 25-40 dB
- Use free tools like Audacity to adjust audio
