## 🛠️ Method 1: OBS + Microsoft Whiteboard (BEST FOR MAC/WINDOWS)

### Step 1: Download & Install Software

**OBS (Open Broadcaster Software)**
```
https://obsproject.com/
- Free
- Works on Mac, Windows, Linux
- Professional quality
- Perfect for screen recording
```

**Microsoft Whiteboard**
```
Mac: App Store
Windows: Microsoft Store (or https://whiteboard.microsoft.com/)
- Free
- Clean white background
- Smooth drawing
- Perfect for this use case
```

### Step 2: Setup MS Whiteboard

```
1. Open Microsoft Whiteboard
2. Create new board
3. Blank white canvas appears
4. You're ready to write!

Tool Bar Options:
├─ Pen (Black) - Main writing tool
├─ Eraser - Fix mistakes
├─ Shape Tool - Draw boxes/arrows
└─ Text Tool - Type text
```

### Step 3: Script Your Video

**Before recording, write a script:**

```
Question 1 Script (2:30 minutes):

[0:00-0:10] Write question
  "Q: What is science?"

[0:10-0:15] Pause

[0:15-0:25] Write answer
  "Answer: B"

[0:25-0:35] Pause

[0:35-1:45] Write explanation
  "Why B is correct:"
  "Science is a systematic"
  "process of:"
  "• Observing"
  "• Questioning"  
  "• Testing"
  "• Learning"
  
  [Draw arrow connecting concepts]

[1:45-2:30] Final summary
  "Science = Systematic thinking"
  [Underline key words]
  
[2:30] End video
```

### Step 4: Setup OBS for Recording

**Open OBS:**

```
1. Click "+" under Sources
2. Select "Display Capture" (Mac) or "Display Capture" (Windows)
3. Choose your monitor
4. Adjust preview to show your Whiteboard app window
5. Click "Start Recording"
```

**OBS Settings (Recommended):**

```
Settings → Output → Recording
├─ Recording Path: /Users/yourname/Desktop/examtool/video/
├─ Format: MP4
├─ Video Encoder: H.264 (Hardware)
├─ Bitrate: 5000-8000 kbps (high quality)
└─ Audio Bitrate: 128 kbps

Settings → Video
├─ Base Resolution: 1920x1080
├─ Output Resolution: 1920x1080
└─ FPS: 30
```

### Step 5: Record Your Video

**Recording Process:**

```
1. Open MS Whiteboard (full screen)
2. Open OBS (in background)
3. Click "Start Recording" in OBS
4. Wait 2 seconds (for audio sync)
5. Start writing on whiteboard
6. Speak your explanation clearly
7. Write everything step by step
8. When done, click "Stop Recording"
9. OBS saves MP4 automatically
```

**During Recording Tips:**
- Speak clearly and slowly
- Write large enough to read
- Use pen steadily (not shaky)
- Leave pauses between sections
- Name the file: `q001-correct.mp4` or `q001-incorrect.mp4`

## 📹 Recording Best Practices

### Audio Quality
```
✅ DO:
- Use clear, slow speaking
- Pause between concepts
- Use natural tone
- Speak into microphone

❌ DON'T:
- Speak too fast
- Have background noise
- Whisper or mumble
- Use robotic tone
```

### Video Quality
```
✅ DO:
- Write large and clear
- Use straight lines
- Organize info logically
- Use colors (black pen, maybe red for highlights)

❌ DON'T:
- Write tiny text
- Scribble messily
- Jump around randomly
- Shake the screen
```

### Pacing
```
Fast-paced video (2 min):
- Q displayed: 10 sec
- Answer shown: 5 sec
- Explanation: 90 sec
- Summary: 35 sec

Slow-paced video (3+ min):
- Same structure, but longer pauses
- More detailed explanation
- More visual aids
```

## 🎬 Step-by-Step Recording Session

### Before Recording
```
1. Close all unnecessary apps
2. Mute phone and notifications
3. Open Whiteboard app (full screen)
4. Open OBS (in background)
```

## 🔧 Troubleshooting

### Problem: Video is dark/hard to see
```
Solution:
1. Check display brightness
2. Check OBS recording quality settings
3. Re-record with better lighting
4. Increase contrast in whiteboard
```

### Problem: Audio is out of sync
```
Solution:
1. Pause 2-3 seconds at video start
2. Record audio and video together (not separate)
3. Use OBS (does sync automatically)
4. Re-record if still out of sync
```

### Problem: Text is blurry
```
Solution:
1. Increase font size (minimum 20pt)
2. Write slower (give OBS time to capture)
3. Use bold fonts
4. Increase bitrate in OBS
5. Re-record in higher resolution
```

### Problem: Takes too long to record
```
Solution:
1. Write faster and speak at normal pace
2. Remove unnecessary details
3. Practice script first
4. Simplify explanations
5. Keep to 2-3 minutes max
```
